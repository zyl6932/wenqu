Ollama 是一个本地运行大语言模型的工具。

Ollama 支持多种模型，包括 Qwen、Llama、Mistral 等。

安装 Ollama 后，可以使用 ollama pull 命令拉取模型，例如：
ollama pull qwen2.5:1.5b

Ollama 默认运行在 http://localhost:11434 端口。

知识库系统通常需要两个模型配合：一个大语言模型用于生成回答，一个嵌入模型用于向量化文档。

常用的嵌入模型包括 all-minilm、nomic-embed-text、bge-m3 等。其中 all-minilm 体积最小（约67MB），适合资源有限的场景。

RAG（Retrieval-Augmented Generation）是一种结合检索和生成的技术。它先从知识库中检索相关文档片段，再将片段作为上下文提供给大模型生成回答。

RAG 的优势在于：答案基于实际文档，减少幻觉；知识可以随时更新，不需要重新训练模型。
