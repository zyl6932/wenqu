# 问渠 使用说明

## 你需要准备

1. **一台电脑**（Windows / macOS / Linux 均可）
2. **网络**（调用 DeepSeek API 需要联网）

## 第一步：安装 Python

1. 打开 [python.org](https://www.python.org/downloads/)
2. 下载 Python 3.10 或更高版本
3. 安装时**勾选 "Add Python to PATH"**

验证安装：打开终端（Windows 按 Win+R 输入 `cmd`），输入：
```
python --version
```
显示 `Python 3.x.x` 即成功。

## 第二步：安装 Ollama

1. 打开 [ollama.com](https://ollama.com)
2. 点击 Download 下载对应系统版本
3. 安装后会自动启动，屏幕顶部会出现 Ollama 图标

拉取向量化模型（必须）：
```
ollama pull bge-m3
```

（可选）拉取本地对话模型：
```
ollama pull qwen2.5
```

## 第三步：下载项目

1. 打开 [github.com/zyl6932/wenqu](https://github.com/zyl6932/wenqu)
2. 点击绿色的 Code 按钮 → Download ZIP
3. 解压到你想要的文件夹，比如 `D:\wenqu`

或者在终端中：
```
git clone https://github.com/zyl6932/wenqu.git
cd wenqu
```

## 第四步：安装依赖

在项目文件夹中打开终端，输入：
```
pip install -r requirements.txt
```

## 第五步：配置 API Key

1. 在项目文件夹中找到 `.env.example`，复制一份改名为 `.env`
2. 用记事本打开 `.env`，找到这行：
   ```
   DEEPSEEK_KEY=sk-your-key-here
   ```
3. 把 `sk-your-key-here` 替换成你的 DeepSeek API Key
   - 申请地址：[platform.deepseek.com/api_keys](https://platform.deepseek.com/api_keys)
   - 新用户通常有免费额度

> 如果你有本地 Ollama 对话模型（qwen2.5 等），可选：
> 改为 `LLM_PROVIDER=ollama`，就完全不需要 API Key 了。

## 第六步：放入文档

把你要问的文档（txt / md / docx / pdf）拖到项目里的 `docs` 文件夹。

## 第七步：启动

在项目文件夹终端中输入：
```
python server.py
```

看到以下输出说明启动成功：
```
  ✓ LLM API Key: 使用 DeepSeek: deepseek-chat
  ✓ 端口检查
  ✓ Ollama 服务
  自检通过 ✓
  服务已启动: http://localhost:8080
```

## 第八步：使用

打开浏览器，访问 **http://localhost:8080**

1. 点击左下角「设置」
2. 点「+ 导入文档」
3. 导入完成后，在输入框输入问题，按 Enter 发送

### 常用操作

| 操作 | 方式 |
|------|------|
| 新建对话 | 左侧栏「+ 新建对话」 |
| 切换深色/浅色 | 设置 → 主题 |
| 调整字号 | 设置 → A+/A- |
| 删文档 | 设置 → 文档列表右边 × |
| 编辑分块 | 点击文档名打开块编辑器 |
| 拖拽上传 | 直接把文件拖到页面任意位置 |

### 手机访问

确保手机和电脑在同一 WiFi，电脑 IP 可在终端输入 `ipconfig` 查看。手机浏览器访问 `http://电脑IP:8080`。
